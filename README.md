В данном проекте реализована игра "Кольцевое движение"
Описание игры: аркадная игра, где игрок управляет объектом-точкой внутри большого кольца. Внутри кольца движутся несколько вражеских объектов-точек. Цель игры - собирать бонусы и избегать столкновения с врагом. Бонусы появляются случайным образом.
Движение врагов происходит с помощью генерации чисел от 0 до 7 - и в зависимости какая цифра выпадет выбирается направление вдижения: вверх, вниз, вправо, влево, и по диагоналям. 
Бонусы также появляютсяв различных местах. 
У главного героя есть три состояния: круг, квадрат и треугольник. В начале двигается круг, при захвате 10 и более бонусов - герой перевоплощается в квадрат двигается далее, а при 21 и более бонусов - треугольник который остается до конца игры. 
Окончание игры и вывод счета возможно при случаях:
-нажатие кнопки ESC
-при столкновении с врагом
